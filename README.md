## Как запустить локально

Проект полностью статический. Можно запустить через любой локальный сервер из корня.

Например, через Python:

```bash
python3 -m http.server 8080
```

После этого открыть в браузере: `http://localhost:8080`

## Пару слов по реализации

- Сборщики и препроцессоры подключать не стал, чтобы не усложнять и не тянуть лишние node_modules — везде чистый HTML, CSS и JS.
- Верстал по БЭМ, стили и скрипты у каждой задачи свои.
